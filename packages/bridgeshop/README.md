## Introducción

BridgeShop es una plataforma de comercio electrónico moderna orientada a TypeScript, construida con GraphQL y React. Diseñada para desarrolladores, ofrece funciones esenciales de comercio dentro de una arquitectura modular y totalmente personalizable, perfecta para construir experiencias de compra a medida de forma rápida y confiable.

## Instalación usando Docker

Puedes empezar con BridgeShop en cuestión de minutos usando la imagen de Docker. Es una excelente manera de iniciarte en BridgeShop sin tener que preocuparte por instalar dependencias o configurar tu entorno de desarrollo.

```bash
curl -sSL https://raw.githubusercontent.com/dp/bridgeshop/main/docker-compose.yml > docker-compose.yml
docker-compose up -d
```
